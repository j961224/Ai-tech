**1. 학습 데이터 셋을 구해야한다!!**

=> 서비스 요구사항으로부터 구해야한다!

=> 종류와 수량도 정해야한다!

**2. 기술 모듈 설계를 한다.**

-> 학습 별로 정답은 AI 모델 별로 입력에 대한 출력 쌍이다!!

**3. 테스트 데이터셋/테스트 방법 준비**

**4. 모델 요구사항 도출**

> * 처리 시간
> > * 처리 시간은 하나의 입력이 처리되어 출력이 나올 때까지의 시간!
> > * 예) 수식 영역 검출의 경우
> * 목표 정확도
> > * 해당 기술 모듈의 정량적인 정확도
> > * 예) 신용카드 인식의 경우

* 목표 qps
